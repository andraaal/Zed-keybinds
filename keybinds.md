# Zed Keybinds

## Panes

Ctrl-Shift-G: Toggle Git panel  
Ctrl-Shift-E: Toggle Project panel  
Ctrl-Shift-D: Toggle Debug panel  
Ctrl-I: Toggle Agent  
Ctrl-Ö: Toggle Terminal panel  

Ctrl-W: Close active pane/file/dock (Ctrl-Shift-W in terminal)

## Splits (File Finder)

Ctrl-I, Ctrl-J, Ctrl-K, Ctrl-L

## Pane Navigation

Ctrl-Shift-I: Move pane focus up  
Ctrl-Shift-K: Move pane focus down  
Ctrl-Shift-L: Move pane focus right  
Ctrl-Shift-J: Move pane focus left  


## Search / Replace

Ctrl-Shift-F: Search project  
Ctrl-F: Search file  
Ctrl-Shift-H: Replace project  
Ctrl-H: Replace file  


## Refactor

Ctrl-7: Comment line  
Ctrl-Shift-7: Comment line and advance  
Ctrl-Alt-7: Block comment  
F2: Rename symbol  


## Navigation

Ctrl-G: Go to line  
Ctrl-B: Go to definition  
Ctrl-Alt-B: Go to implementation  
Ctrl-Shift-B: Go to type definition  
Ctrl-P: Find file  
Ctrl-T: Find symbol in project  
Ctrl-O: Find symbol in file  

Ctrl-Alt-Left: Go back  
Ctrl-Alt-Right: Go forward  


## Misc

Ctrl-Space: Show completions  
Alt-Enter: Show code actions  
Ctrl-Q: Show code documentation  
Alt-Shift-F: Format code  
Shift Shift / Ctrl-Shift-P: Open command palette  
Ctrl-Shift-T: Reopen closed item  
Ctrl-Shift-R: Run  


## Diagnostics

Ctrl-Shift-M: Open diagnostics panel  
F8: Next diagnostic  
Shift-F8: Previous diagnostic  

## Selection

Ctrl-D: Select next occurrence  
Ctrl-E: Expand selection  
Ctrl-L: Select line  
Ctrl-Shift-A: Select all matches  

## Line Operations

Alt-Up: Move line up  
Alt-Down: Move line down  
Alt-Shift-Up: Duplicate line up  
Alt-Shift-Down: Duplicate line down  
Alt-Y: Delete line  
Ctrl-J: Join lines  

## Cursors

Alt-Click: Add cursor  
Ctrl-Shift-Up: Add cursor above  
Ctrl-Shift-Down: Add cursor below  


## Basic

Ctrl-N: New file  
Ctrl-S: Save  
Ctrl-A: Select all  
Ctrl-X: Cut  
Ctrl-C: Copy  
Ctrl-V: Paste  
Ctrl-Z: Undo  
Ctrl-Shift-Z: Redo
